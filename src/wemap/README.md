## Dependencies

- Install via npm or yarn:

```bash
$ npm install
# install with yarn
$ yarn install
```

## Usage

- In a Node.js application.

```js
const WeMap = require('wemap')

const text = 'Hello World!'
const weMapText = WeMap(text)

console.log(weMapText)

```

- In a modular environment.

```js
import WeMap from 'wemap'
```

### Build

```bash
yarn build

```
